What This System Does

  • Automatically reads COI emails from your mailbox
  • Extracts key fields (insured, holder, dates, coverage)
  • Matches certificates to tenants and vendors
  • Tracks expiration dates automatically
  • Sends reminder emails at 60/30/7/1 days before expiration
  • Flags issues for your review
  • Detects renewals and cancels old reminders
  • Stores certificates in Postgres database
  • Provides web dashboard for review and approval
  • Supports bulk roster import via CSV

For Meredith's Team

This pilot system is designed for Knight Real Estate's property management workflow.

  • Import your current tenant/vendor roster
  • Process incoming COI emails
  • Accept, assign, or reject certificates
  • Monitor expiring certificates
  • Stay compliant with lease requirements
  • Reduce manual tracking overhead
